# Подготовка main к github

1. Для продолжения разработки и сохранения всех текущих артефактов создается ветка dev, а main очищается от всего, что не относится напрямую к исследованию и пайплайну.
Ветки dev и main впредь имеют независимую, отдельную историю.

2. Из main убираются:
- Все из data, за исключением того, что использовалось напрямую в исследовании для возможности воспроизводимости результатов (в т.ч. отдельные experiments)
- docs/
- external/
- logs/
- memory_bank/
- results/figures
- term-paper/
- term-paper_2/
